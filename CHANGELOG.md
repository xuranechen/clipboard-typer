# Changelog

## 0.1.0

- 新增 Python + Tkinter 桌面应用骨架。
- 支持读取剪贴板、文本预览、倒计时输入、停止输入。
- 支持字符间隔、行间隔、Tab 策略、最大长度、输入模式和预设模板。
- 新增快速模式 `fast`，适合本地普通窗口快速输入。
- 首次启动自动生成并修复 `config.json`。
- 增加输入失败重试与明确错误提示。
- 增加清空预览、清空剪贴板按钮。
- 状态栏显示当前模式、延迟、目标窗口和点击坐标。
- `start.bat` 增加 `start.log`，打包脚本增加版本信息。
- 支持配置保存、输入前确认、密码模式、完成后清理预览/剪贴板。
- 支持可选全局快捷键和 Esc 停止。
- 新增 Windows PyInstaller 构建脚本。
- 新增目标窗口选择：下拉窗口列表、拾取相对坐标、自动激活并点击后输入。
